# تتبَّع — منصة التشغيل المتكاملة للشركات

صفحة الهبوط (Landing Page) الرسمية لمنصة **تتبَّع**: منصة تشغيل أعمال متكاملة للشركات ومكاتب الاستشارات المهنية، تجمع كل الأعمال اليومية في نظام واحد مدعوم بالذكاء الاصطناعي.

## نبذة عن المنصة

تتبَّع ليست مجرد أداة لإدارة المهام، بل منصة احترافية لإدارة وتشغيل الأعمال بالكامل، وتضم تسع منظومات متكاملة:

- إدارة العملاء
- إدارة المشاريع
- إدارة المهام
- إدارة الملفات والمستندات
- الصلاحيات المتقدمة
- التقارير والمتابعة
- أتمتة العقود وعروض الأسعار
- الذكاء الاصطناعي

**دورة عمل المهمة:** منفّذ ← مراجِع ← مدير المشروع (الاعتماد النهائي)، مع إشعارات تلقائية وسجل تغييرات كامل عند كل انتقال.

**الفئة المستهدفة:** مكاتب الاستشارات المهنية والمحاسبية، والشركات الاستشارية والخدمية، والفرق التي تدير عدداً كبيراً من العملاء والمشاريع والمهام.

## محتويات المشروع

| المسار | الوصف |
|--------|-------|
| `site/index.html` | صفحة الهبوط الكاملة (HTML + CSS + JavaScript في ملف واحد) |
| `site/assets/` | الشعارات (SVG) والمكتبات الخارجية (Swiper) |
| `extracted/` | ملفات الشعار الأصلية |

## التشغيل محلياً

الموقع صفحة ثابتة (Static)، يكفي فتح الملف في المتصفح:

```
site/index.html
```

## التقنيات

- HTML5 / CSS3 / JavaScript (بدون أُطر عمل)
- تصميم عربي بالكامل (RTL) بخط Tajawal
- تصميم متجاوب (Responsive) مع واجهة حجز عرض توضيحي تفاعلية
